## Features

- Fullstack App with **admin panel**.
- **CRUD operations** with posts and comments.
- Create an account and **leave a comment** under the post.
- Admin can **post, delete posts and delete unwanted comments**.


## Tech Stack

- **React** (React Router)
- **Express**
- **REST API**
- **Context API**
- **CSS Modules**
